/tg/ Station 13 - Modules - Types

code/modules/admin/view_variables/debug_variable_appearance.dm

/image/appearanceAn alias datum that allows us to access and view the variables of an appearance by keeping certain known, yet undocumented, variables that we can access and read in a datum for debugging purposes. Kindly do not use this outside of a debugging context.
/proc/build_virtual_appearance_vars Makes a var list of /appearance type actually uses. This will be only called once.
/proc/debug_variable_appearance debug_variable() proc but made for /appearance type specifically
/proc/locate_appearance_variable manually locate a variable through string value. appearance type needs a manual var referencing because it doesn't have "vars" variable internally. There's no way doing this in a fancier way.
/proc/vv_get_header_appearance Shows a header name on top when you investigate an appearance
/proc/get_appearance_vv_summary_name Makes a format name for shortened vv name.
